## 딥보이스 및 보이스 피싱 탐지 앱
- 통화 종료 후 딥보이스 또는 보이스 피싱 여부를 탐지 후 알림을 제공하는 어플리케이션  
## 개발 기간
- 23.10 ~ 11 (1개월)  
## 개발 인원
- 6명: 한혜주(모델), 김동민(앱, 서버), 이승주(기획), 이유진(앱, 서버), 송수영(모델)  
## 개발 내용
- 딥보이스 탐지 기술 개발
  - 실제 음성과 스푸핑 음성 구분
